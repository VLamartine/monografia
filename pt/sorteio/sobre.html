Sobre o site

O objetivo desse site é disponibilizar uma ferramenta de sorteios aleatórios que seja transparente e verificável. O método de sorteio, descrito abaixo, garante que, dada as mesmas entradas (sementes + itens na mesma ordem), o resultado será sempre o mesmo

Método de sorteio

  1. É informado qual o tipo de semente deseja utilizar
    Caso seja utilizada uma semente dos faróis de aleatoriedade disponíveis
    • A semente utilizada é o próximo pulso gerado depois que o sorteio é programado
    Caso a semente seja fornecida manualmente
    • É utilizado o valor fornecido, ou caso esteja em branco, é o utilizado o timestamp do momento que o sorteio é programado
  2. Os itens que devem ser ordenados são informados, separados um por linha
  3. Cada item é concatenado da forma: número da linha+item+semente. Formando uma sentença
  4. Cada sentença é então, codificada utilizando SHA256 como método
  5. Por fim, a sequência de códigos em SHA256 é ordenada lexicograficamente
