About this site

The purpose of this site is to make available a random draw tool, transparent and verifiable. Our draw method, described below, guarantees that, for the same input (seed + items in the same order), the result will always be the same.

Draw method

  1. The user enters which seed origin they desire
    If the origin is one of the available randomness beacons
    • The seed will be the first pulse after the draw is programmed
    If the seed is one manually inserted
    • The input valued is used. If there is no input value, the timestamp of the moment that the draw is programmed is used instead
  2. The User enters the items that they want ordered, one per line
  3. Each item is then concatenated in the following order: line number+item+seed. Forming a string
  4. Each string is encrypted, using the SHA256 method.
  5. And at the end, the SHA256 strings are ordered alphabetically
